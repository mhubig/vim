## Quick start

0. Clone this repository:

    ```
    $ git clone https://github.com/mhubig/vim.git ~/.vim
    ```

0. Create some links:

    ```
    $ ln -s ~/.vim/vimrc ~/.vimrc
    $ ln -s ~/.vim/gvimrc ~/.gvimrc
    $ ln -s ~/.vim $XDG_CONFIG_HOME/nvim
    ```

0. Start Vim and install the plugins with `:PlugInstall`
